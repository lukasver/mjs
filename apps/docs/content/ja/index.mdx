# 「麻雀スターズ」へようこそ

![MJSバナー](/static/images/banner.png)

## ソーシャルゲームの次世代進化

**麻雀スターズ**は、最先端の**Web3ソーシャル麻雀P2Pゲームプラットフォーム**により、デジタルゲーム空間を再定義しようとしています。この革新的なプラットフォームは、画期的なAIハイブリッドシステムを利用しており、プレイヤーは自分の戦略を学習し模倣するAIアバターを作成およびトレーニングできます。これにより、**オフライン時でも24時間365日**、グローバルな麻雀ゲームプレイに継続的に参加できます。

## 麻雀スターズをユニークにするもの

### 🤖 AI搭載ゲーム
- **トレーニング可能なAIキャラクター**: あなたのAIはあなたのプレイスタイルを学習し、あなたのゲームプレイデータでトレーニングできます
- **24時間365日ノンストップ流動性**: あなたがオフラインでもAIがプレイします—制限を設定して継続的に獲得できます
- **ハイブリッドAIプレイ**: 自分自身でプレイすることと、AIに任せることをシームレスに切り替えることができます


### 🌐 Web3統合
- **Unityベースのマルチプラットフォームアプリ**: すべてのデバイスで一貫したゲーム体験
- **簡単なソーシャルログイン**: 組み込みの暗号ウォレット統合によるシンプルで安全なアクセス
- **NFTマーケットプレイス**: キャラクター、コレクタブル、NFTをアンロックしてゲームプレイをパーソナライズできます

### 🎮 ソーシャルゲームの卓越性
- **6億人以上のターゲット市場**: 世界中の膨大な麻雀プレイヤーベースを活用
- **ソーシャルプレイとコミュニティ**: 友達とプレイし、マッチをホストし、独自の麻雀サークルを育てましょう
- **実績のある専門知識**: 10年間の麻雀ゲーム開発経験に基づいて構築
