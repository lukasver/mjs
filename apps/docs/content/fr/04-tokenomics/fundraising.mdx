## Stratégie de lancement du jeton

### Phase 1 : TGE (T4 2025)
- Lancement sur les principaux DEX avec une liquidité initiale de 1 M$
- Mise en œuvre d’un mécanisme de jalonnement pour les premiers utilisateurs
- Lancement des fonctionnalités de gouvernance communautaire

### Phase 2 : Inscriptions sur les CEX
- Cibler les bourses de niveau 1 pour une accessibilité accrue
- Développer les partenariats avec les teneurs de marché
- Améliorer la liquidité sur plusieurs plateformes

### Phase 3 : Extension des utilitaires
- Intégration complète du marché NFT
- Fonctionnalités de gouvernance avancées
- Mise en œuvre du partage des revenus
